Pengertian Ilmu Komputer

Ilmu Komputer adalah bidang studi yang berfokus pada pemahaman, pengembangan, dan penerapan teknologi komputasi. Bidang ini mencakup berbagai aspek seperti analisis algoritma, pengembangan perangkat lunak, pemrograman, desain sistem komputer, dan teknologi yang mendukung komputasi modern. Ilmu Komputer tidak hanya mempelajari cara kerja komputer, tetapi juga mengeksplorasi bagaimana teknologi dapat digunakan untuk memecahkan masalah dalam berbagai aspek kehidupan.

Tugas 1: Desain Jaringan Kabel dan Nirkabel

Langkah-langkah yang perlu dilakukan:

  1. Lihat Gambar 12 pada Buku Bahan Ajar.
  2. Gunakan aplikasi Diagram https://app.diagrams.net/.
  3. Buat diagram sesuai petunjuk.
  4. Pastekan link file diagram di bagian ini.

Tugas 7: Meresume Diagram Routing dan Pendalaman Materi tentang Routing Dynamic

Langkah-langkah yang perlu dilakukan:

  1. Pelajari materi tentang routing.
  2. Buat resume tentang routing.
  3. Buat diagram model routing interkoneksi minimal 3 router (gunakan aplikasi diagram).
  4. Pastekan link file resume dan diagram di sini.
  5. Cari referensi yang lengkap dan buat pembahasan tentang Dynamic Routing (EGP dan IGP).

Cari referensi untuk topik berikut dan buat pembahasan serta presentasinya:

  • EGP - Kelompok 1: BGP, Kelompok 2: BGP
  • IGP - Kelompok 3: RIP, Kelompok 4: IGRP, Kelompok 5: OSPF, Kelompok 6: OSPF, Kelompok 7: EIGRP, Kelompok 8: EIGRP

Tugas 9: Perancangan Arsitektur Jaringan Client-Server

Tugas 10: Membuat Tabel Daftar DNS

Langkah-langkah yang perlu dilakukan:

  3. Jenis domain yang harus dibuat: .com, .net, .or.id, ac.id, .sch.id, go.id, co.id.
  4. Pastekan link file spreadsheet di sini.
