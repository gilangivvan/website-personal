Tugas 7: Meresume Diagram Routing dan Pendalaman Materi tentang Routing Dynamic

Langkah-langkah yang perlu dilakukan:

  1. Pelajari materi tentang routing.
  2. Buat resume tentang routing.
  3. Buat diagram model routing interkoneksi minimal 3 router (gunakan aplikasi diagram).
  4. Pastekan link file resume dan diagram di sini.
  5. Cari referensi yang lengkap dan buat pembahasan tentang Dynamic Routing (EGP dan IGP).

Cari referensi untuk topik berikut dan buat pembahasan serta presentasinya:

  • EGP - Kelompok 1: BGP, Kelompok 2: BGP
  • IGP - Kelompok 3: RIP, Kelompok 4: IGRP, Kelompok 5: OSPF, Kelompok 6: OSPF, Kelompok 7: EIGRP, Kelompok 8: EIGRP
